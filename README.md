## Instructions

Prerequisites
node installed

- to run the web app

```sh
$ git clone https://github.com/GiadaSimonetti/lootcheck
$ cd lootcheck
$ npm install
$ npm start
```
Your browser should open on localhost:3000, if it doesn't go directly to localhost:3000.
